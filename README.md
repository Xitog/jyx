# Jyx
A simple text editor for developers in Python/Tkinter

# Comparison with other editors
This table presents some common features of text editors.

|                     | Notepad | Gedit |  Jyx  |
|---------------------|:-------:|:-----:|:-----:|
| Syntax highlighting |    no   |  yes  |  yes  |
| Statistics          |    no   |  yes  |  yes  |
